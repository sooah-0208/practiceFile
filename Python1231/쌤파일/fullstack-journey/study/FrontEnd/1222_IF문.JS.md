# 🧠 IF문 정리

## 📌 기본 구조
```js
if (조건) {
  // ✅ 참이면 실행
} else {
  // ❌ 조건이 참이 아니면 실행
}
```

```js
if (true) {

}
```

- 🔁 if문의 반대 ⇒ else
- ⚠️ else는 반드시 if와 함께 사용

---

## 🚫 NOT 연산자 (!)
```js
if (!true) {
  console.log('참');
} else {
  console.log('거짓');
}
```

📤 출력값
```
거짓
```

- ❗ `!` = 부정
- ✂️ if문의 조건이 하나일 경우 `{}` 생략 가능

```js
if (a) console.log(a);
```

---

## 🔍 비교 연산자
- `>`  : 크다
- `>=` : 크거나 같다
- `<`  : 작다
- `<=` : 작거나 같다
- `%`  : 나머지

---

## 🔗 IF문 여러 개 사용

### ✅ AND (&&)
```js
if (a > 0 && b > 0) {

}
```

### 🔀 OR (||)
```js
if (a > 0 || b > 0) {

}
```

---

## ⏭️ jump (continue)
- ⛔ continue를 만나면 아래 코드를 건너뜀
- ✨ 생략하고 싶을 때 사용

```js
for (let i = 0; i < 10; i++) {
  if (i === 0) {
    continue;
  }
  if (i % 2 === 0) {
    console.log(i);
  }
}
```

```js
// 📝 i가 0이면 아래 코드를 실행하지 않고 다음 반복으로 넘어감
```

---

## ⚡ 삼항 연산자
```js
a = (조건) ? 참일 때 : 거짓일 때;
```

```js
for (let i = 0; i < 9; i++) {
  a = (i % 2 === 0) ? a + 'true' : a + 'false';
  console.log(a);
}
```

```js
// 1️⃣ a는 true 또는 false 문자열이 추가됨
// 2️⃣ 조건: i % 2 === 0
// 3️⃣ 참이면 a + 'true', 거짓이면 a + 'false'
```

---

## 🔄 삼항 연산자를 IF문으로 변환
```js
for (let i = 0; i < 9; i++) {
  if (i % 2 === 0) {
    a + 'true';
  } else {
    a + 'false';
  }
  console.log(a);
}
```
