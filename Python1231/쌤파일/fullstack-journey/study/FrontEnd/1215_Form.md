# 💡 form 형태 배우기
[Form Tag](../HTML_TAG_LIST.md) : 여기 5번
- 줄 바꿈: `<p>` or `<div>`로 항목 나눔
- `required`: 내용을 입력하지 않으면 오류메세지가 뜨는 기능
- -`checked`: `radio, chkeckbox`등 체크박스가 뜨는 요소에서 한 항목을 미리 선택해놓도록 보이게 만드는 기능
- -`disabled`: `radio, chkeckbox` 요소에서 선택이 불가하도록 함
  
## 1.  `file `: 기본설정은 1개 파일만 선택 가능 -> multiple로 다중선택 가능하게 함  
- 확장명 제어(png/jepg...) -> accept="png" or "image/*" 
- style="display: none; 이거 넣으면 `파일선택, 선택된 파일 없음`이 사라짐
=>>BUT! lable에 for / input에 id 꼭 넣어야 `파일`글자 클릭했을 때 파일창이 켜짐
- file을 전달하기 위해선 `form`에 action="/action_page_binary.asp" method="post" enctype="multipart/form-data" 를 입력해줘야함

  
## 2. 숫자
- min="" / max="" 설정 가능
- step="" 으로 숫자 띄어넘길 단위 정할 수 있음(5->5 10 15...로 선택가능)

# ⭐ 1차 답
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<style>
    *{box-sizing: border-box;}
    html, body {width: 100%;
        height: 100%; margin: 0;}
    .row {
        display: flex;
        align-items: center;
        width: 100%;
        padding: 6px;}
    .row>input{width: 100%;}
    form{padding: 10px 5px;}
    label{width: 50px;}
    input{width: calc(100%-50px);}
    button{background-color: rgb(89, 124, 221);
        color: rgb(0, 0, 0);
        width: 100%;
        height: 25px;
        padding: 0 5px;
        border-radius: 5px;
        border: none;
        outline: none;}
</style>
<body>
    <form>
        <div class="row">
            <label for="txt">글자: </label>
            <input id="txt" name="txt" type="text" placeholder="글자를 입력해주세요.">
        </div>
        <div class="row">
            <label for="number">숫자: </label>
            <input type="txt" id="number" type="text" placeholder="10부터 입력해주세요.">
        </div>
        <div class="row">
            <label for="file">파일: </label>
            <input type="file" id="file" autocapitalize="off">
        </div>
        <button type="button">업로드</button>
    </form>
</body>
</html>
```

- 여기서 `label for=""`는 라벨(글자/숫자)부분을 클릭했을 때도 입력창으로 넘어가도록 설정하는 코드.
=>input의 id로 연결됨. / 다른 input id를 전부 동일 시키면 어떤 라벨을 눌러도 같은 입력창에 들어가게됨.
- `placeholder="내용"` 내용이 입력칸에 회색글자로 들어감. 입력하면 사라지는 글자.
- autocapitalize="off"` 그동안 입력했던 내용을 보여주지 않을 때 사용. (기본은 on으로 설정되어있음)
