# 🧩 Flowgorithm

## 🖥️ 프로그램 설명
- 💡 코딩을 **시각적으로 보여주는 프로그램**
- 🪟 레이아웃 창을 누르면 **코드 + 콘솔을 동시에 확인 가능**


---


## 📦 변수 (Variable)

### ✏️ 변수 선언
```js
let / var / const = 변수 선언
```

```js
const a = 변수 배정
```

- 🔒 `const` : 값 변경 ❌
- 🔓 `let / var` : 값 변경 ⭕

### ▶️ 실행 흐름
- `let / var`
- 입력 → 재생 ▶️ → 콘솔창에 **변수값 출력**

---

## 🔀 제어문 (IF문)
```js
if = 제어 → 조건식 입력
```

- 조건이 참이면 실행
- 거짓이면 실행 안 함

---

## 🧵 문자열 & 템플릿 리터럴

```js
`${}`
```

- 📎 문자열 연결 역할 (`&` 처럼 사용)

```js
"문자열"
```

- ⚠️ 문자열은 **무조건 "" 사용**

---

## 🧩 함수 (Function)

### 📞 함수 호출이란?
- 📏 주 프로그램이 너무 길어질 때
- ✂️ 일정 코드 묶음을 **함수로 분리**
- 🏷️ 이름(name)을 지정
- 🔁 주 프로그램에서 **함수 이름으로 호출**

### 흐름 정리
```
함수 정의 → 함수 이름 지정 → 주 프로그램에서 호출
```

- 🧠 Flowgorithm = 시각적 알고리즘 도구
- 📦 변수 = 값 저장

- 🧵 `${}` = 문자열 연결
- 🧩 함수 = 코드 정리 & 재사용
