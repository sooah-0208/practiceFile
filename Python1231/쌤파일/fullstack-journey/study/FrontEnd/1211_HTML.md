## 🛠 Visual Studio Code 기본

* **파일 확장자명은 반드시 입력해야 함**
* 확장자가 없으면 기본적으로 **텍스트 파일(text file)** 로 인식됨

### 자주 사용하는 확장자

* `파일명.md` → Markdown 문서
* `파일명.html` → HTML 문서
* `파일명.py` → Python 스크립트
* `파일명.txt` → 일반 텍스트 파일

## 📄 HTML 기초

### 1️⃣ 태그(Tag)란?

* 태그는 `< >`로 감싸진 **HTML의 기본 구성 요소**
* 시작 태그와 종료 태그로 구성됨

---

### 📌 자주 쓰는 태그 정리

* `h1 ~ h6` : 제목 태그 (숫자가 작을수록 글씨가 큼)

  ```html
  <h1>제목</h1>
  <h2>소제목</h2>
  ```

* `div` : 영역(레이아웃) 구분용 태그

  <img width="219" height="258" alt="div 구조 이미지" src="https://github.com/user-attachments/assets/228e3072-2f50-4c02-802a-63407732b352" />

* `p` : 문단(텍스트 영역)

* `li` : 리스트 항목

* `ol` : 순서가 있는 목록 (자동 번호)

* `ul` : 순서가 없는 목록

---

### 🔗 네비게이션 관련 태그

* `nav` : 주요 링크 묶음 (메뉴 영역)
* `a` : 링크 생성 태그

  * 주요 속성: `href` (이동할 주소)

```html
<nav>
  <a href="index.html">홈</a>
  <a href="about.html">소개</a>
  <a href="contact.html">문의</a>
</nav>
```

## 🎨 style (CSS 꾸미기)

* HTML 요소를 꾸미는 역할
* `선택자 { 속성: 값; }` 형태로 작성

### 예시

```html
<style>
  aside {
    width: 30%;
    padding-left: 15px;
    padding-right: 15px;
    margin-left: 80px;
    margin-right: 40px;
    float: right;
    background-color: rgb(135, 235, 188);
  }

  header {
    background-color: rgb(225, 202, 247);
    margin-right: 40px;
  }
</style>
```
