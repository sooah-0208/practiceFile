# 1️⃣ `<nav>` + `<a>` +`<ul>``<li>`
메뉴 화면에 사용하는 코드 => 이동 메뉴 영역이라고 알려주는 태그
해당 위치로 이동시켜준다

## ✅ <a> (링크) — 이동용
```
<nav>
  <a href="#home">Home</a>
</nav>
```
- 페이지 이동 / 스크롤 이동
- 새 페이지 열기 가능
- 내비게이션 메뉴에는 정석
- SEO, 접근성 👍 (SEO란? 검색엔진 최적화: 내 사이트 구조를 검색엔진이 이해하기 쉽게 만드는 것)
- 👉 메뉴라면 기본은 a

# 2️⃣ `<a href="#home">`
### ✅ **“id가 home인 위치로 이동하라”**
즉 id="home"을 만들어줘야함(보통 section코드에 넣어줌, div도 가능)

# ✅ “진짜 홈 화면”으로 가고 싶다면?
1. 새 페이지
`<a href="index.html">Home</a>`
2. 루트로 이동
`<a href="/">Home</a>`
3. 현재 페이지 맨 위
`<a href="#">Home</a>`

# 3️⃣ `<ul>`->`<li>`
메뉴는 **목록**이기때문에 ul 리스트 형식으로 만들어야 의미에 맞음
- 스크린 리더로 읽어냈을 때 사용자가 nav의 li몇개구나 -> 아 메뉴가 4개있구나 라고 이해할 수 있음
- div로 쓰면 메뉴인지/ 몇 개인지/ 구조가 어떤지 알 수 없음
- 스타일링 하기도 쉬워짐(모바일 메뉴 만들기도 좋음)

# 4️⃣ form -> lable -> input
input엔 type, name 필수
보통 로그인화면 등에 사용
action은? form을 보낼 곳이 있다면 넣어줌
`<form action="send.php" method="post">` => “submit 버튼을 누르면 이 입력값들을 send.php로 전송해라”
정렬시 
1) 부모인 form에게 `text-aling: (center)`를 넣어주면 됨(input은 inline 요소라서)
2) margin조절 `input {display: block; / margin: 10px auto;}`
3) **flex**:`form {display: flex; / flex-direction: column; / align-items: center;}`

*input에 넣으면 input안 글씨가 중앙정렬됨*
